# 快速开始

**知几喵** 是一款功能强大、体验出众的第三方 AI 客户端，与 CatWiki 深度对接。配合 **OpenClaw Gateway**，即使在复杂的内网环境下，您也能获得安全、稳定的移动端访问体验。

> **获取方式**：iOS 用户请前往 **App Store** 搜索 “**知几喵**” 即可下载安装。

## 1. 在知几喵中增加 OpenClaw 管理

OpenClaw 是专为私有化部署设计的 AI 网关。在知几喵中集成 OpenClaw，可以实现更强大的内网穿透与多环境管理。

### 1.1 配置 OpenClaw Gateway

1. **进入 Gateway 管理**: 在知几喵“设置”页面找到 **OpenClaw** 栏目，进入 **Gateway 管理**。
    <div align="center">
      <img src="/images/zhijimiao/openclaw/8.png" width="200" alt="进入设置" />
    </div>

2. **添加连接**: 点击右上角的“+”号，填写网关设置（地址、端口、Token 等）。
    <div align="center">
      <img src="/images/zhijimiao/openclaw/7.png" width="200" alt="添加 Gateway" />
    </div>

3. **连接成功**: 成功连接后可进入网关详情页进行管理。
    <div align="center">
      <img src="/images/zhijimiao/openclaw/6.png" width="200" alt="网关详情" />
    </div>
