# 知几喵 APP 介绍

**知几喵** 是一款功能强大、体验出众的 AI 智能客户端，支持 iOS 平台。

> **获取方式**：iOS 用户请前往 **App Store** 搜索 "**知几喵**" 即可下载安装。

## 核心功能

- 🤖 **多模型管理**：支持添加多个 OpenAI 兼容的 AI 模型服务，快速切换使用
- 💬 **智能对话**：原生流式输出、Markdown 渲染、代码高亮，提供流畅的对话体验
- 🔗 **OpenClaw 网关**：内置 OpenClaw Gateway 管理，支持通过网关安全访问内网部署的 AI 服务并进行对话
