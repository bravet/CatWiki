# 快速上手

本文将指引您完成知几喵 APP 的下载安装，添加 AI 模型并开始对话。

## 1. 下载安装

- **iOS**：前往 **App Store** 搜索 "**知几喵**"，点击下载安装。

  <img src="/images/logo.png" width="200" alt="logo" />
</div>
## 2. 添加模型

1. 打开知几喵，进入 **"设置"** 页面。
2. 点击 **"添加模型"**，选择 **"OpenAI 兼容"** 类型。
3. 填写连接信息：
   - **API 地址**：模型服务的 API 端点地址
   - **API Key**：模型服务提供的密钥
   - **模型名称**：要使用的模型名称
4. 点击 **"保存"**，完成添加。

> [!TIP]
> 知几喵支持同时添加多个模型服务，可在对话中快速切换使用不同模型。

## 3. 开始对话

模型添加成功后，即可创建新对话，直接与 AI 模型进行交互。

> [!TIP]
> 如果您的 AI 服务部署在内网环境，请参考 [OpenClaw 对接指南](/zhijimiao/openclaw-quick-start) 配置网关穿透后再添加模型。
