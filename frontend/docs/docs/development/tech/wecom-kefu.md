# 企业微信客服

> [!NOTE]
> **“微信客服”是企业微信提供的一种全渠道对外服务能力**。与“自建应用”不同，它直接连接外部微信用户，通过配置特定的“客服帐号”来开启与 CatWiki 的 AI 对接。

企业微信客服通过标准的 Webhook 回调方式集成。与普通的“智能机器人”不同，客服消息采用 XML 协议，且由于平台限制，必须通过异步 API 接口进行回复。

## 推荐场景
- **全渠道对外客服**：通过微信扫码、视频号、公众号等入口，让普通微信用户直接调起 AI 客服。
- **售后支持**：客户无需安装应用，直接在微信聊天框即可获得专业的售后知识解答。
- **售前咨询**：与微信生态无缝连接，是企业接触外部客户最直接的 AI 应用载体。


## 1. 企业微信开放平台配置

### 1.1 获取客服凭据
1. 登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/loginpage_wx)。
2. 进入 **“应用管理” -> “应用” -> “微信客服”**。
3. 记录下网页底部展示的 **“企业ID” (CorpID)**。
4. 进入 **“API 接口”** 部分，获取或生成 **Token** 和 **EncodingAESKey**。
5. 获取 **客服 Secret**。

### 1.2 配置回调地址
1. 在“微信客服”管理页面的 **“接收消息”** 部分，配置回调 URL。
2. 格式为：`https://您的域名/v1/bot/wecom-kefu?site_id={site_id}`
3. *注意：请将 `{site_id}` 替换为您在 CatWiki 后台的实际站点 ID。*
4. 配置完成后点击验证。

---

## 2. CatWiki 后台配置

1. 进入 CatWiki 后台 **“站点设置” -> “AI 机器人”**。
   ![AI机器人集成](/images/screenshots/8.png)
2. 选中 **“企业微信客服”**，开启开关。
3. **填入配置**：
   - **CorpID**：填入企业 ID。
   - **Secret**：填入客服 Secret。
   - **Token**：填入从企微获取的 Token。
   - **EncodingAESKey**：填入从企微获取的 EncodingAESKey。
4. 点击 **“保存”**。

---

## 3. 技术特性
- **流式同步频率**：约为 1.0s / 次。
  - **说明**：受限于微信客服异步接口的调用频率，系统会在此间隔内收集生成的回复片段，并批量调用发送接口，以平衡回复速度与接口流控。
- **协议机制**：采用标准的 XML 协议进行消息加解密。
- **回复方式**：异步调用 `kf/send_msg` 接口。
